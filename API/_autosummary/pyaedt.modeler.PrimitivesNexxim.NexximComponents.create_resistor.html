create_resistor¶

NexximComponents.create_resistor(compname=None, value=50, xpos=0, ypos=0, angle=0, use_instance_id_netlist=False)[source]¶

Create a resistor.

Parameters
compnamestr, optional

Name of the resistor. The default is None.

valuefloat, optional

Resistance in ohms. The default is 50.

xposfloat, optional

Position on the X axis. The default is 0.

yposfloat, optional

Position on the Y axis. The default is 0.

anglefloat, optional

Angle rotation in degrees. The default is 0.

use_instance_id_netlistbool, optional

Whether to use the instance ID in the net list. The default is False.

Returns
int

ID of the resistor.

str

Name of the resistor.
