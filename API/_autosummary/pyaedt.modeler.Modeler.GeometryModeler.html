pyaedt.modeler.Modeler.GeometryModeler¶

class pyaedt.modeler.Modeler.GeometryModeler(parent, is3d=True)[source]¶

Manages the main AEDT Modeler functionalities for geometry-based designs.

Parameters
parent

Inherited parent object.

is3Dbool, optional

Whether the model is 3D. The default is True.

Returns
bool

True when successful, False when failed.
