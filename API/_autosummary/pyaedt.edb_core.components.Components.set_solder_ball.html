set_solder_ball#

Components.set_solder_ball(component='', sball_diam='100um', sball_height='150um')[source]#

Set cylindrical solder balls on a given component.

Parameters:
component_namestr or EDB component

Name of the discrete component.

sball_diamstr, float, optional

Diameter of the solder ball.

sball_heightstr, float, optional

Height of the solder ball.

Returns:
bool

True when successful, False when failed.

Examples

>>> from pyaedt import Edb
>>> edbapp = Edb("myaedbfolder")
>>> edbapp.core_components.set_solder_ball("A1")
