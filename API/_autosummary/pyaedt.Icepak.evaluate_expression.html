evaluate_expression¶

Icepak.evaluate_expression(expression_string)¶

Evaluate a valid string expression and return the numerical value in SI units.

Parameters
expression_stringstr

A valid string expression for a design property or project variable. For example, "34mm*sqrt(2)" or "$G1*p2/34".

Returns
float

Evaluated value for the string expression.
