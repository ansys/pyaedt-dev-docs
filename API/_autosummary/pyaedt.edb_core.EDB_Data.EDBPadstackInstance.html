pyaedt.edb_core.EDB_Data.EDBPadstackInstance#

class pyaedt.edb_core.EDB_Data.EDBPadstackInstance(edb_padstackinstance, _pedb)[source]#

Manages EDB functionalities for a padstack.

Parameters
edb_padstackinstance
_pedb

Inherited AEDT object.

Examples

>>> from pyaedt import Edb
>>> edb = Edb(myedb, edbversion="2021.2")
>>> edb_padstack_instance = edb.core_padstack.padstack_instances[0]
