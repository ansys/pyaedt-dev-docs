pyaedt.modules.Boundary.FarFieldSetup¶

class pyaedt.modules.Boundary.FarFieldSetup(app, component_name, props, component_type, units='deg')[source]¶

Manages Far Field Component data and execution.

Examples

in this example the sphere1 returned object is a pyaedt.modules.Boundary.FarFieldSetup >>> from pyaedt import Hfss >>> hfss = Hfss() >>> sphere1 = hfss.insert_infinite_sphere() >>> sphere1.props[“ThetaStart”] = 45 >>> sphere1.delete()
