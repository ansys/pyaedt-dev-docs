create_polygon¶

EdbLayout.create_polygon(main_shape, layer_name, voids=[], net_name='')[source]¶

Create a polygon based on a list of points and voids.

Parameters
main_shape

Shape of the main object.

layer_namestr

Name of the layer on which to create the polygon.

voidslist, optional

List of shape objects for voids. The default is``[]``.

net_namestr, optional

Name of the net. The default is "".

Returns
bool

True when successful, False when failed.
