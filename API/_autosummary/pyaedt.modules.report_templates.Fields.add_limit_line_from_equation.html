add_limit_line_from_equation¶

Fields.add_limit_line_from_equation(start_x, stop_x, step, equation='x', units='GHz', y_axis=1)¶

Add a Cartesian Limit Line from point lists. This method works only in graphical mode.

Parameters
start_xfloat

Start X value.

stop_xfloat

Stop X value.

stepfloat

X step value.

equationstr

Y equation to apply. Default is Y=X.

unitsstr

X axis units. Default is “GHz”.

y_axisstr, optional

Y axis. Default is “Y1”.

Returns
bool
